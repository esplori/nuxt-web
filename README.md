# nuxt-web

## 项目简洁：

这是一个用nuxt开发的项目  

演示地址：[www.dsiab.com](https://www.dsiab.com)

![avatar](https://gitee.com/wilkwo/nuxt-web/blob/master/assets/img/site.jpg)

## 选择nuxt的原因：

1.nuxt开发的项目能被搜索引擎收录

2.相比其他语言，nuxt更简单，跟vue开发一样简单，会前端就可以开发

3.能正常接入统计跟广告代码


## 启动应用

```bash
# 安装依赖
$ npm install

# 本地启动在 localhost:3000
$ npm run dev

# 部署到服务器，先build，再执行start启动
$ npm run build
$ npm run start

# 生成静态项目
$ npm run generate
```
